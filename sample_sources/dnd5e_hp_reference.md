# D&D 5e HP Quick Reference

Section: Hit points by class

- A paladin uses a d10 Hit Die.
- At level 1, a paladin starts with 10 hit points plus the character's Constitution modifier.
- At each paladin level after level 1, add either a d10 roll plus Constitution modifier or the fixed value 6 plus Constitution modifier. The table or player must choose rolled or fixed HP.

Section: Constitution modifier

- Constitution 16 or 17 gives a +3 modifier.

Section: Dwarf ancestry note

- A plain dwarf ancestry does not by itself change hit points in this demo reference.
- If a table uses a dwarf option that grants extra hit points, the character build must state that option before the calculation.
